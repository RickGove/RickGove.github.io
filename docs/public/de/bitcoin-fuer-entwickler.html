Bitcoin für Entwickler

Mit Bitcoin können erstaunliche Dinge geschaffen und gängige Probleme gelöst werden.

Das einfachste aller Zahlungssysteme

Wenn Zahlungen nicht mit automatischer Rechnungsstellung verbunden sein müssen, ist das Akzeptieren von Geld so einfach wie das Senden von Bitcoins - es muss nur ein Link oder QR-Code angezeigt werden. Diese einfache Konfiguration ist für quasi jeden Nutzer machbar und kann die Bedürfnisse vieler Kunden erfüllen. Aus buchhalterischer Sicht ist es besonders gut geeignet um Gemeinkosten zu verringern und die Transparenz zu erhöhen.

Viele Third-Party-APIs

Es gibt viele Third-Party-Anbieter für Zahlungsabwicklungen, die APIs bereitstellen. Sie müssen Bitcoins nicht auf Ihrem Server speichern und sich um die damit verbundenen Sicherheitsvorkehrungen kümmern. Die meisten dieser APIs ermöglichen Ihnen außerdem, Rechnungen abzuwickeln und Ihre Bitcoins zu wettbewerbsfähigen Kosten in Ihre lokale Währung umzutauschen.

Sie können Ihre eigene Bank sein

Wenn Sie keine Third-Party-APIs nutzen, können Sie einen Bitcoin-Node direkt in Ihre Anwendungen integrieren, um dadurch Ihre eigene Bank und Zahlungsanbieter zu werden. Wenn Sie sich der damit verbunden Verantwortung bewusst sind, können Sie beeindruckende Systeme schaffen, die Bitcoin-Transaktionen nach Ihren Vorlieben verarbeiten.

Bitcoin-Adressen zur Verfolgung von Rechnungen

Bitcoin erzeugt für jede Transaktion eine eindeutige Adresse. Wenn Sie ein Zahlungssystem mit Rechnungsstellung erstellen wollen, müssen Sie lediglich für jede Zahlung eine Bitcoin-Adresse generieren und überwachen (Sie sollten niemals die gleiche Adresse für mehr als eine Transaktion benutzen).

Für Sicherheit sorgt hauptsächlich der Client

Die Sicherheit wird zum Großteil durch das Protokoll sichergestellt, sodass PCI-Konformität überflüssig ist. Betrugsprävention kann auf das Überwachen einer einzigen Variable reduziert werden: Den Bestätigungs-Score. Abgesehen davon ist die sichere Aufbewahrung von Bitcoins eine Sache der Sicherung der Wallet und der Verwendung von HTTPS oder anderen gesicherten Protokollen, um Zahlungsaufforderungen an Kunden zu senden.

Eine neue Welt voller Möglichkeiten

Bitcoin ermöglicht es, neue kreative Online-Dienste zu gestalten, die früher aufgrund von finanziellen Einschränkungen nicht möglich waren. Dies beinhaltet Trinkgeld-Systeme, automatisierte Zahlungslösungen, dezentralisierte Crowdfunding-Dienste, öffentliches Asset-Tracking, zeitgebundenes Zahlungsmanagment ("time-locked payment management"), vertrauensunabhängige Treuhanddienste ("low-trust escrow services"), Micro-Payment-Kanäle und mehr.
