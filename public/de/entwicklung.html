Bitcoin Entwicklung

Finden Sie weitere Informationen zur gegenwärtigen Spezifikation, zur Software und zu Entwicklern.

  • Dokumentation
  • Entwickler-Community
  • Bitcoin Core Mitwirkende
  • Weitere freie Software-Projekte

Bitcoin ist freie Software und jeder Entwickler kann zu dem Projekt beitragen. Alles was Sie benötigen finden Sie im GitHub Repository. Bitte lesen und befolgen Sie den in der README beschriebenen Entwicklungsprozess, liefern Sie Codes von guter Qualität und respektieren Sie alle Richtlinien.

Diskussionen über die Entwicklung finden auf GitHub und über die bitcoin-dev E-Mail-Liste statt. Weniger formale Diskussionen über die Entwicklung finden bei irc.freenode.net #bitcoin-core-dev (Webinterface, Logs) statt.

Dokumentation

Falls Sie daran interessiert sind, mehr über die technischen Details von Bitcoin zu erfahren und wie man vorhandene Tools und APIs verwendet, dann wird empfohlen bei der Dokumentation für Entwickler zu beginnen.

Entwickler-Community

Die folgenden Chatrooms und Websites bieten Diskussionen über die Bitcoin-Entwicklung. Lesen Sie bitte die Verhaltensregeln bevor Sie sich beteiligen.

  • IRC Channel #bitcoin-core-dev auf freenode.
  • Bitcoin StackExchange
  • BitcoinTalk Development & Technical Discussion Forum
